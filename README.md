# Veículo Autônomo para Depósito de Objetos
Este projeto está vinculado a disciplina de Projeto Integrador 1, da Universidade de Brasília, ministrada pelo Prof. Dr. Lui Txai Calvoso Habl. Tem por objetivo o desenvolvimento de um **veículo autônomo** capaz de seguir trajetórias predefinidas e depositar um objeto do tamanho, proporção e fragilidade de um ovo médio em uma caixa acolchoada ao término da trajetória, sendo operado por uma interface gráfica que retornará o tempo e a trajetória do veiculo.

## Objetivos

- **Trajetórias e Navegação**:
  O carrinho deverá seguir **três diferentes trajetórias**:
  1. **Primeira Trajetória**: Indicação visual com fita isolante no piso.
  2. **Segunda Trajetória**: Indicação visual com fita isolante intermitente.
  3. **Terceira Trajetória**: Indicação de percurso baseada apenas em instruções dadas pelos professores (sem marcação visual).
  
- **Depósito do Objeto**: O carrinho deverá **transportar um objeto frágil (ovo médio)** e entregá-lo de forma segura em uma caixa acolchoada no final da trajetória.


## Etapas do Projeto

### 1. **Desenvolvimento Conceitual**
- Características gerais do carrinho e suas funcionalidades.
- Descrição detalhada do hardware e software envolvidos.
- Análise do consumo energético.

### 2. **Levantamento de Requisitos**
- Brainstorming
- Organização de Requisitos 
- Separação de requisitos por área de desenvolvimento
- Levantamento de Requisitos de baixo nível 

### 3. **Planejamento**
- Definição do cronograma de desenvolvimento.
- Estimativas de custos e orçamento.

### 4. **Prototipagem e Testes**
- Protótipos de hardware e simulação das trajetórias.
- Testes de desempenho (tempo e distância).
- Ajustes e validação de cada parte do sistema.

### 5. **Documentação**
- Desenvolvimento contínuo de artigo científico, documentando cada etapa do projeto.

### 6. **Entrega Final**
- Apresentação do projeto final, incluindo um vídeo de demonstração do funcionamento do carrinho.
- Relatório final com descrição detalhada dos resultados experimentais e análise do desempenho do carrinho.

## Tecnologias Utilizadas

- **Hardware**:
  - Sensores de movimento (IMU, giroscópio, etc.).
  - Atuadores (motores, rodas).
  - Microcontroladores e componentes eletrônicos.

- **Software**:
  - Interface gráfica para programação das trajetórias.
  - Banco de dados para registro e análise dos dados coletados.
  - Linguagens de programação: C++, Python, ou outras adequadas para controle de hardware.

- **Comunicação**:
  - Comunicação sem fio (Wi-Fi, Bluetooth, etc.).

## Contribuições

O desenvolvimento do projeto será realizado por uma equipe multidisciplinar, com a contribuição de **engenharias da FCTE** (Faculdade de Ciências e Tecnologia da Engenharia).

Cada membro da equipe será responsável por diferentes áreas, incluindo hardware, software, estrutura, e documentação do projeto.
